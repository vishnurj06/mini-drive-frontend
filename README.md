Mini Drive is a full-stack, hierarchical cloud storage application built with pure Vanilla JavaScript. It replicates the core functionality of Google Drive, allowing users to upload, organize, and securely share their files and folders.



\## ✨ Key Features

\* \*\*Secure Authentication:\*\* JWT-based login, strict domain validation, and EmailJS OTP password recovery.

\* \*\*Hierarchical Storage:\*\* Create nested folders and navigate seamlessly using dynamic breadcrumbs.

\* \*\*Granular Sharing Engine:\*\* Share files and folders with specific users, granting 'View' or 'Edit' permissions.

\* \*\*Access Requests:\*\* Native request flows for unauthorized users trying to access restricted content.

\* \*\*Admin Master Control:\*\* A dedicated, secure dashboard for platform administrators to manage all user data across the platform.

\* \*\*File Previews:\*\* Built-in viewers for Images, PDFs, Audio, Video, and Office Documents (PPT/DOC).



\## 🚀 Running Locally

Because this frontend is built without heavy frameworks, running it locally is incredibly fast:

1\. Clone this repository: `git clone https://github.com/vishnurj06/mini-drive-frontend.git`

2\. Open the folder in VS Code.

3\. Launch `index.html` using the \*\*Live Server\*\* extension.



\*Note: This frontend requires the \[Mini Drive Backend] -> ([link](https://github.com/vishnurj06/mini-drive-backend)) to function locally.\*
